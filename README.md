# 🛒 E-Commerce Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing an e-commerce dataset using SQL, Python (EDA), and Power BI to generate business insights.

---

## ⚙️ Tools & Technologies Used

* SQL (MySQL / PostgreSQL)
* Python (Pandas, Matplotlib, Seaborn)
* Power BI
* Jupyter Notebook

---

## 📊 Key Features

### 🔹 Executive Summary Dashboard

* Total Revenue: 16M
* Total Orders: 99K
* Total Customers: 96K
* Average Order Value: 160.58

---

### 👥 Customer Analysis

* Repeat Customers: 3K
* New Customers: 93K
* Repeat Rate: 3.12%
* Top States by Customers

---

### 📦 Product Analysis

* Total Products Sold: 113K
* Top Category: Health & Beauty
* Avg Rating: 4.09
* Top Revenue Products

---

### 💳 Payment & Order Analysis

* Total Payments: 104K
* Avg Installments: 2.85
* Payment Success Rate: 97.02%
* Payment Methods Distribution

---

## 📂 Project Structure

* SQL scripts for data extraction
* Python notebook for EDA
* Power BI dashboards for visualization

---

## 🚀 Insights

* Most revenue comes from Health & Beauty category
* Very high payment success rate (~97%)
* Low repeat customer rate indicates growth opportunity
* Majority orders are successfully delivered

---

## 📌 Conclusion

This project demonstrates end-to-end data analysis workflow:
data cleaning → analysis → visualization → business insights.
